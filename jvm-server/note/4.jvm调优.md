#### jstack检测cpu高
* 得到java进程pid
* 查看cpu占用高线程
    * top -H -p 17850
* 转换线程ID
    * printf "%x\n" 17880  -> 45d8
* 定位cpu占用线程
    * jstack 17850|grep 0x45d8 -A 300
